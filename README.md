# College Managemnet Portal (MERN Stack) App

**Front-End** - React </br>
**Back-End** - Node.js, Express.js & MongoDB

## SetUp
### For server side </br>
Clone the server folder which has the following dependecies
body-parser
    * express
    * connect-mongo
    * cookie-parser
    * cors
    * express-session
    * mongoose
    * multer
    * passport
    * passport-local
